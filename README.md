# ❤️ Heart Disease Prediction System

##  Overview

This project is an end-to-end machine learning application that predicts the likelihood of heart disease using patient data.

##  Model

* Algorithm: Random Forest Classifier
* Accuracy: ~87%
* Balanced precision & recall ensuring reliable predictions

## ⚙️ Features

* Data preprocessing & feature scaling
* Model training & evaluation
* Real-time prediction via Streamlit UI
* Deployed on cloud for public access
